# DeepFapar — 深度学习驱动的 fAPAR 遥感估算框架

**项目名称** DeepFapar | **技术栈** TensorFlow / Keras | **目标** 从无人机多源数据中估算光合有效辐射吸收比例

## 概述

DeepFapar 是一套基于深度学习的工具包，用于从无人机遥感数据中估算 fAPAR（光合有效辐射吸收比例）及相关生物物理量。项目基于 TensorFlow/Keras 构建，实现了一个可配置的前馈神经网络，能够学习多源输入特征与目标生物物理变量之间的复杂非线性映射关系。

## 研究动机

fAPAR 等生物物理变量与光谱反射率波段、植被指数和气象输入之间存在高度非线性的关系。基于树的集成方法通过分段常数逼近来捕获这些关系，而神经网络则提供了一种连续、平滑的函数逼近方式，能够更好地表征底层的物理过程——尤其是在异质性农业景观中，特征间的交互作用往往是微妙且多层次的。

DeepFapar 面向需要灵活、可复现框架的研究人员，使其无需从零搭建流程即可完成冠层尺度性状估算模型的训练、评估和部署。

## 网络架构

默认网络采用两隐藏层全连接架构：

```
输入层（10 个特征）
    → 全连接层（64 个神经元，ReLU 激活）
    → 全连接层（32 个神经元，ReLU 激活）
    → 输出层（1 个神经元，线性激活）
```

关键架构设计选择：

- **隐藏层使用 ReLU 激活函数** — 引入非线性能力，同时避免了 Sigmoid 网络中常见的梯度消失问题。
- **输出层使用线性激活** — 允许无界的回归目标，适合连续型生物物理变量的预测。
- **采用 Adam 优化器** — 为每个参数提供自适应学习率，在此规模的表格数据上比标准 SGD 收敛更快。
- **使用 MSE 损失函数** — 直接优化均方误差，与农业遥感领域通行的 RMSE 评估标准保持一致。

所有架构参数——网络深度、宽度、激活函数、优化器和训练轮数——均可通过 `config/NN_params.py` 在外部配置，便于快速尝试更深或更宽的网络结构。

## 输入特征

模型使用涵盖大气条件、冠层结构和土壤属性的十个特征：

- **大气类**：`logpre`（降水量）、`logvpd`（饱和水汽压差）、`logppfd`（光合有效光子通量密度）
- **冠层类**：`chi`（聚集指数）、`loggpp`（总初级生产力）、`AI`（干旱指数）
- **土壤/环境类**：`soc`（土壤有机碳）、`logtp`（全磷）、`logtn`（全氮）、`gtmp`（生长温度）

所有特征在输入网络前均进行最小-最大归一化处理，归一化参数来自训练集，以防止数据泄露。

## 训练过程

训练采用基于检查点的策略：

1. 网络按可配置的轮数进行训练（默认 100 轮）。
2. 每一轮训练结束后，将完整的模型状态保存为 `.keras` 检查点文件。
3. 全部训练完成后，找出验证损失最小的轮次。
4. 加载该轮次对应的检查点作为最终模型——而非最后一轮的模型。

这种方式起到了隐式早停的作用，既不会过早中断训练，又保留了完整的训练历史供后续诊断分析。

## 评估与诊断

流程生成两组可视化诊断结果：

- **训练曲线** — 将损失和 MAE 随轮数的变化分别绘制在训练集和验证集上，便于检测过拟合或欠拟合。
- **预测散点图** — 在测试集上绘制预测值与观测值的对比图，标注 R²、RMSE、RRMSE、MAE 和 Nash-Sutcliffe 效率系数。

这些产出以 PNG 图片形式保存，同时生成包含逐轮指标、超参数记录和最终模型性能的 Excel 工作簿。

## 适用场景

DeepFapar 适用于以下情况：

- 输入与目标之间的关系被怀疑具有高度非线性。
- 有足够的训练数据（数百至数千个样本）。
- 研究人员希望探索不同的网络深度和宽度，而无需重写流程代码。
- 可复现性和检查点级别的可追溯性对科学报告很重要。

对于数据量较小或模型可解释性优先的场景，配套项目 **ForestFapar**（基于随机森林）可能是更合适的选择。
